# Music note detector using stm32f411ret6
## Description
This project will capture frequencies and identifying musical notes.
e.g.  The frequency is 440, the note will be A4.
## Components Required
- Nucleo-f411re.
- 20x4 LCD Display with i2c.
- MAX9812.
- Breadboard.
- Connecting Wires.
